# Difference Calculator

[![Maintainability](https://api.codeclimate.com/v1/badges/b6c4a297ef4ad3b91629/maintainability)](https://codeclimate.com/github/Vanger-Li/frontend-project-lvl2/maintainability)  ![Build Status](https://github.com/Vanger-Li/frontend-project-lvl2/workflows/Node.js%20CI/badge.svg)

Difference Calculator is a cli utility compares two configuration files and shows a difference.

## Install ##

Clone this repo and run:
```
make install
make link
```
