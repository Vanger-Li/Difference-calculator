Difference Calculator is a cli utility compares two configuration files and shows a difference.

## Install ##

Clone this repo and run:
```
make install
make link
```

## Program launch
### Comparison of flat files:

```sh
$ gendiff filepath1.json filepath2.json
```

